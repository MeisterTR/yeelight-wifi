▸ setColorTemperature(temperature, [effect], [time])

Will change the color temperature of the Yeelight

setColorTemperature(temperature: string, effect: [string], time: [number]): Promise
Parameters
temperature (string) is the target color temperature. The type is integer and range is 1700 ~ 6500 (k).
effect ([string] (default 'smooth') ) support two values: 'sudden' and 'smooth'. If effect is 'sudden', then the color temperature will be changed directly to target value, under this case, the third parameter 'duration' is ignored. If effect is 'smooth', then the color temperature will be changed to target value in a gradual fashion, under this case, the total time of gradual change is specified in third parameter "duration".
time ([number] (default 1000) ) time specifies the total time of the gradual changing. The unit is milliseconds. The minimum support duration is 30 milliseconds.
Returns
Promise: will be invoked after successfull or failed send
Example
setColorTemperature(5000);
setColorTemperature(5000, 'sudden');
setColorTemperature(5000, 'smooth', 1000);

▸ setScene(params)

This method is used to set the smart LED directly to specified state. If + the smart LED is off, then it will turn on the smart LED firstly and then apply the specified command.

setScene(params: array): Promise
Parameters
params (array) can be "color", "hsv", "ct", "cf", "auto_dealy_off".
"color" means change the smart LED to specified color and brightness.
"hsv" means change the smart LED to specified color and brightness"
"ct" means change the smart LED to specified ct and brightness.
"cf" means start a color flow in specified fashion.
c"auto_delay_off" means turn on the smart LED to specified brightness and start a sleep timer to turn off the light after the specified minutes. "val1", "val2", "val3" are class specific.
Returns
Promise: will be invoked after successfull or failed send
Example
setScene(['color', 65280, 70]);
setScene(['hsv', 300, 70, 100]);
setScene(['ct', 5400, 100]);
setScene(['cf', 0, 0, '500,1,255,100,1000,1,16776960,70']);

▸ setAdjust(action, prop)

This method is used to change brightness, CT or color of a smart LED without knowing the current value, it's main used by controllers.

setAdjust(action: string, prop: string): Promise
Parameters
action (string) the direction of the adjustment. The valid value can be:
'increase': increase the specified property
'decrease': decrease the specified property
'circle': increase the specified property, after it reaches the max value, go back to minimum value
prop (string) the property to adjust. The valid value can be:
'bright': adjust brightness.
'ct': adjust color temperature.
'color': adjust color. (When 'prop' is 'color', the 'action' can only be 'circle', otherwise, it will be deemed as invalid request.)
Returns
Promise: will be invoked after successfull or failed send
Example
setAdjust('increase', 'bright');

▸ startColorFlow(count, action, flowExpression)

This method is used to start a color flow. Color flow is a series of smart LED visible state changing. It can be brightness changing, color changing or color temperature changing.

startColorFlow(count: number, action: string, flowExpression: string): Promise
Parameters
count (number) is the total number of visible state changing before color flow stopped. 0 means infinite loop on the state changing.
action (string) is the action taken after the flow is stopped.
0: means smart LED recover to the state before the color flow started.
1: means smart LED stay at the state when the flow is stopped.
2: means turn off the smart LED after the flow is stopped.
flowExpression (string) is the expression of the state changing series.
Returns
Promise: will be invoked after successfull or failed send
Example
startColorFlow(4, 2, '1000, 2, 2700, 100, 500, 1, 255, 10, 500, 2, 5000, 1');
